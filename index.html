Assman

Frictionless Web Asset Management for ASP.NET MVC. Easily manage dependencies and compile scripts and stylesheets.

Overview

  • Manages dependencies between scripts and automatically includes the dependencies when you request a script on the page
  • Scripts and Styles consolidated by convention (you define the convention that works for your project), and thus requires minimal configuration/maintenence
  • Minifies your scripts and stylesheets
  • Manages the inclusion/rendering of Scripts and Stylesheets so that they can be rendered in the most optimal section of the page
  • Contains a hook to support including resources from a CDN when your site uses one
  • Extensible plugin architecture to support extending the library (dotLess plugin already exists, CoffeeScript and Sass on the way)
  • Optimizes development experience in Debug mode, optimizes website performance in Release mode
  • Does not require any tools to be installed

Getting started

Install instructions for:

  • ASP.NET MVC 3
  • ASP.NET MVC 2

Download the source

You can clone the project with Git by running: 

$ git clone git://github.com/andyalm/assman
